\abstractCS{Tato bakalářská práce se zabývá návrhem a implementací knihovny pro automatizaci testů verifikace komunikace průmyslové komunikace. Knihovna funguje nad testovacím frameworkem MSTest. Práce také zkoumá existující open-source knihovny pro protokoly ModbusTCP a Ethernet/IP a jejich vhodnost implementace do knihovny. Na závěr práce zhodnocuje výsledné řešení z pohledu projektového řízení.}
\abstractEN{This bachelor's thesis is focused on design and implementation of test automation library for verification of industry fieldbus communication. This library works as add-in to testing framework MSTest, which enables connection with Azure DevOps. Azure DevOps then handles the automation of the tests. This thesis also researches existing open-source libraries for fieldbuses ModbusTCP and Ethernet/IP and evaluates their suitability for implementation into the library. Lastly the thesis also evaluates the solution from the project managment view.}
